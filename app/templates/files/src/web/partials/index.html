Binarta starter template

This is a very basic template to help you create a new Binarta app very quickly.
It only has a homepage, a blog and a contact page.

Good luck with your new project!

i18n

Use the i18n directive to make a text translatable.

Example: 

<span i18n code="test.i18n.example" default="This text can be translated" ng-bind="var"></span>

For allowing full wysiwyg functionality, use the editor attribute.

Example: 

<i18n code="test.i18n.example.2" default="This <i>text</i> can have some <b>formatting</b>." editor="full" ng-bind-html="var|trust"></i18n>

Images

Use the bin-image directive in combination with the img element to make it editable.

Example: 

<img bin-image="test.image.example.1">
